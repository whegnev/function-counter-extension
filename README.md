# Function Counter

**Автор:** Лежнина Александра, М3105
**Версия:** 1.0.0

## Описание
Плагин для VS Code, который подсчитывает количество функций, операторов if и циклов в коде и отображает статистику в реальном времени.

## Возможности
- Подсчет функций для разных языков программирования
- Подсчет операторов if
- Подсчет циклов for и while
- Отображение статистики в статус-баре

## Установка
1. Клонируйте репозиторий
2. Перейдите в папку проекта
3. Выполните `npm install`
4. Нажмите F5 для запуска в режиме отладки

## Использование
Откройте файл с кодом - статистика автоматически появится в правом нижнем углу.

## Поддерживаемые языки
- JavaScript/TypeScript
- Python
- Java
- C#
- C/C++
